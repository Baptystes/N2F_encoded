🦄 N2F Autopilot

Glisse le bouton dans ta barre de favoris, puis clique dessus quand tu es sur N2F.

N2F Autopilot remplit automatiquement tes dépenses dans N2F à partir de ton relevé d'activité validé sur SIMUS. Il détecte les jours de télétravail et de déplacement, te montre un aperçu, et crée les dépenses (TT, IKMD, abonnement transport) pour toi : fini la saisie une par une.

🦄 N2F Autopilot
  1. Affiche ta barre de favoris si elle est masquée : Ctrl+Maj+B (Windows) ou Cmd+Maj+B (Mac). C'est la rangée de raccourcis sous la barre d'adresse.
  2. Glisse le bouton violet ci-dessus jusque dans cette barre, puis relâche : un favori 🦄 N2F Autopilot apparaît.
  3. Ouvre N2F sur la page d'une note de frais, puis clique sur le favori : le panneau N2F Autopilot s'ouvre.

Un simple clic sur le bouton ci-dessus ne fait rien : il faut le glisser dans la barre de favoris.
